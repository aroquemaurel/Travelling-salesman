Description détaillée

Objet des erreurs.

Toutes les chaines de caractères des erreurs. Si une variable vaut NULL, l'erreur n'est pas présente, sinon elle sera affiché. nbErrors est le nombre d'erreur, si celui-ci =0 alors le programme peut fonctionner correctement
